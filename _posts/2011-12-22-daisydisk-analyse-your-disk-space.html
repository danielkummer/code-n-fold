Time to start a new series: Tools you can use.

Since years i've been using DiskInventory X to analyse and clean my disk.
The tool itself displays the disk space usage as cushion map.
Sadly the tool itself is quite old by now and doen't see active development and updates anymore.

My new recommendation is DaisyDisk. A nice and fast tool which displays the disk space as pie chart. My experiences so far are just positive. Although the full version isn't free its still cheap and worth the money.
